This is my 1.2.5 project. This is a fruit-catching game in which you are supposed to catch the apples and avoid the poisonous fruit, in a Minecraft theme. This game features a leaderboard with the top 3 scores, the score and the 3 lives that you have each attempt, and progressive difficulty that makes the game hard as your score goes up. The player uses arrow keys to control the basket on the bottom and catch as many fruit as possible!
